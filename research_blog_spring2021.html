Weeks 1-5: Progress Report

Feb 10, 2021 - Mar 17, 2021

Objective: Create two new website applications: DecodonCalc3, and DecodonCalcSet.

Back into the research, I was tasked with creating two new website applications that are based off of the original decodoncalc2 website application on TCNJ’s server DecodonCalc2

DecodonCalc3:

This website application will add in the functionality of letting the user input ratios for each selected amino acid and submitting them. By doing so, the application will calculate the ratios of the degenerate codons needed to most closely match the given ratio of amino acids that will be synthesized from such degenerate codons. This takes into account for the preferable codons for each organism as well as the coupling ratios for each DNA base.

DecodonCalcSet:

This website application will add in the functionality of showing the users the degenerate codon sequence, the amino acids that it will code for, and scoring for the maximum cardinality subset, as well as the minimal cardinality superset of the resulting degenerate codon sequence calculated. This will be calculated for amino acids that result in more than 1 degenerate codon sequence, and will return the subset and superset for degenerate codon sets for each n-1 degenerate codons. For example, a selection that results in 3 degenerate codons will show results for the subset and superset for 2 degenerate codons, and 1 as well.

Progress: I’ve been switching between both website applications every week.

DecodonCalc3 LINK

I was able to integrate dynamic text fields in javascript so that the number of ratio input fields created match the number of amino acids selected by the user. Resetting or not selecting any amino acids will also clear any input text fields. A calculate button is also created to calculate the ratios of degenerate codons for the desired ratios of the amino acids, as well as a validation function to check for valid inputs and ratios that total to 100%. Now I need to integrate a java program that inputs the degenerate codon set and user inputted ratios to calculate the resulting ratios.

DecodonCalcSet LINK

I needed to create a new database specifically for this website application. Similar to the original database, each organism table still includes the count of degen codons, binary code for amino acids, and degen codon subset(s). However, now we included two extra attributes, the maximum cardinality subset and minimal cardinality superset. We are also planning on adding score attributes that refer to the codons preferable for specific organisms. After creating the database, now I need to figure out how I could recursively query the two resulting bicodes (subset and superset) to find the matching bicode and further querying the subsets of subsets and supersets of supersets.
